Customizable quick links dropdown in the WordPress admin bar. Works on both frontend and backend. Configure your own shortcuts to any admin page.

== Description ==

**Stop clicking through endless menus to get where you need to go!**

NB Quick Links adds a customizable "Quick" dropdown to your WordPress admin bar that appears on **both the frontend AND backend**. Configure shortcuts to any admin page, plugin settings, or external URL.

Part of the **NetBound Tools** suite.

= Features =

* **Works Everywhere** - Same quick links on frontend and backend admin bars
* **Fully Customizable** - Add, remove, reorder any links you want
* **Drag & Drop Reordering** - Organize your links visually
* **Section Separators** - Group related links together
* **Emoji Icons** - Make links easy to identify at a glance
* **DIVI Detection** - Auto-adds DIVI links if theme is active
* **Frontend Page Actions** - Edit, delete, password-protect, or disable pages right from the frontend

= Frontend-Only Features =

When viewing a page/post on the frontend, you also get:

* **Shortcode Detection** - See which shortcodes are used on the current page
* **Page Actions Menu** - Quick edit, DIVI builder, trash, draft toggle, password protect

= Default Links Included =

* All Posts / Pages / Media / Comments / Plugins / Users
* New Post / Page / Media / Plugin / User
* Widgets / Menus / Themes / Customizer
* Settings / Permalinks / Site Health / Updates
* DIVI options (if DIVI theme active)

All links are editable - remove what you don't use, add what you do!

== Installation ==

1. Upload the `nb-quick-links` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. The "Quick" dropdown appears immediately in your admin bar
4. To customize: Go to **NetBound Tools > ⚡ Quick Links**

== Frequently Asked Questions ==

= Where do I configure the links? =

Go to **NetBound Tools > ⚡ Quick Links** in your WordPress admin.

= Can I add external URLs? =

Yes! Enter any full URL starting with `http://` or `https://`.

= How do I create a separator? =

Set the icon to `---` and the label to something like `─── Section Name ───`

= Does this slow down my site? =

No. The plugin only loads for logged-in administrators.

= Will it work with my theme? =

Yes! It uses the standard WordPress admin bar which works with all themes.
